Descripción general de la distribución y sus características


Usa una distribución GNU/Linux es una distribución de software basada en el núcleo Linux que incluye determinados paquetes de software para cumplir con los requisitos impuestos.

Por lo general están compuestas, total o mayoritariamente, de software libre, aunque a menudo incorporan aplicaciones o controladores propietarios.

Manjaro Linux ofrece de serie el soporte multimedia, una robusta solución para la detección de hardware, soporte para múltiples versiones del kernel y un proceso de instalación gráfico.

El modelo de desarrollo rolling release significa que el usuario no necesita reinstalar el sistema en cada lanzamiento. La instalación se hacía por medio de un sencillo instalador gráfico llamado Thus, desde la versión 15.09 se usa CALAMARES como instalador o mediante una interfaz semi gráfica de terminal.

La gestión de paquetes es manejada por pacman pero añade una interfaz gráfica para búsqueda e instalación de paquetes, Pamac en el caso de las ediciones basadas en GTK y Octopi en las basadas en Qt. Ambos incluyen un notificador de actualizaciones disponibles y son compatibles con AUR.

Existen versiones de 32-bits y 64-bits y es binariamente compatible con Arch Linux.

Los repositorios son manejados con su propia herramienta denominada Boxlt diseñada al estilo git.

En el año 2019, Manjaro tenía disponibles los siguientes escritorios:

  1. Distribuciones oficiales: Xfce, KDE y GNOME.
  2. Distribuciones comunitarias: Openbox, Mate, Cinnamon, Awesome, LXDE, LXQt, BSPWM, Budgie e I3.
  3. Distribuciones para ARM v8: Odroid N2 KDE, Odroid N2 Lxqt, Rockpro 64 Minimal, Pinebook I3, Pinebook KDE, Pinebook Lxqt, Raspberry PI 3 Lxqt, Raspberry PI 3 Minimal, Rock PI 4 Minimal, Rockpro64 Minimal, Rockpro64 KDE, Rockpro64 Lxqt.
  4. Única distribución que soporta 32 bits: Xfce.
  5. Manjaro Architect, que permite realizar una instalación del sistema completamente personalizable usando la línea de comandos de linux.

Dentro de sus características también podemos destacar:

  1. Velocidad, potencia y eficiencia
  2. Acceso a las versiones más modernas del software
  3. Un modelo de desarrollo Rolling-release
  4. Acceso a al repositorio de usuarios de Arch (AUR)

2.4 Licencia

Una licencia de software es un contrato entre el licenciante y el licenciatario del programa informático, para utilizarlo cumpliendo una serie de términos y condiciones establecidas dentro de sus cláusulas, es decir, es un conjunto de permisos que un desarrollador le puede otorgar a un usuario en los que tiene la posibilidad de distribuir, usar o modificar el producto bajo una licencia determinada.

El tipo de licencia de Manjaro es GPL


GPL es una licencia de derecho de autor ampliamente usada en el mundo del software libre y código abierto, y garantiza a los usuarios finales la libertad de usar, estudiar, compartir y modificar el software.

Tiene dos propósitos: declarar que el software cubierto por esta licencia es libre, y protegerlo de intentos de apropiación que restrinjan esas libertades a nuevos usuarios cada vez que la obra es distribuida, modificada o ampliada.

Repositorio de la distribución

Un repositorio (o "repositorio") es básicamente una ubicación de almacenamiento, generalmente un servidor en Internet, desde el que puede instalar software. Los usuarios de Manjaro, de forma predeterminada, utilizarán el repositorio oficial de Manjaro. Se accede a estos repositorios a través de los " Manjaro Mirrors "

Qué es un mirrror?

Es un servidor que almacena una copia actualizada de los paquetes software de una distribución, un ejemplo de un repositorio es Arch User Repository (AUR)

Y un espejo se sicroniza a través de servicios Rsync como por ej: //ftp.tsukuba.wide.ad.jp/manjaro (Asia / Japón), siempre se recomienda sincronizar desde la ubicación más cercana

Conclusiones

En este trabajo he aprendido que la distro de Manjaro Linux nos sirve para poder utilizar Arch Linux pero de una forma muchisimo más sencilla, debido a que tenemos una interfaz gráfica y no hay que aprenderse solo comandos.

Al realizar la búsqueda de paquetes preinstalados conocí bastante sobre interfaces gráficas, paquetes que llevan a cabo la seguridad, el encriptado (cryptsetup), el sonido (pulseaudio), la distribución de archivos(systemd) ... .

También aprendí a como instarlo, en una máquina virtual, y usarlo a través de los vídeos explicativos anteriormente enlazados, se podría decir que tiene una instalación sencilla, con una detección del hardware del equipo automática. Y para finalizar tiene una velocidad, rendimiento y potencia excelentes.

En la búsqueda de los repositorios conocí lo que es un mirror y que sin ellos no se podría emplear un repositorio.

Manjaro tiene una interfaz gráfica y una distribución del escritorio muy similar a Windows y tiene un terminal más orientada a Linux
